# Agents

Custom specialized agents for the hustle plugin.

## Creating Agents

Each `.md` file defines an agent with specialized capabilities.

**Example: `helper.md`**
```markdown
---
name: helper
description: Helps with common development tasks
---

You are a helpful development assistant specialized in...
[agent instructions here]
```

## Usage

Agents are invoked via the Task tool with the agent name.
